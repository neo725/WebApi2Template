# Introduction
這個專案是提供 RD 部門做 WebApi 專案開發使用的範本，內容依照本中心所規範的格式所撰寫
主要開發工具為 Visual Studio 2017
目標 .NetFramework 為 .NetFramework 4.6.1
預設使用 Dapper 為資料庫套件
使用 IIS Express 掛載即可
